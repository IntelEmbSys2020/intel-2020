# 2020年Intel杯中心板

## 需求概览

- 可读取遥控器的6路PWM信号
- 可输出复现6路PWM信号
- 输出PWM可切换模式
  - 原遥控器数据直接输出
  - 上位机命令转换输出
  - 全离线状态紧急降落模式
- 对光流模块I2C接口进行转接（本地供电）
- 可读取上位机UART数据(CH340转换)
- 可输出两路PWM控制云台
- 预留一路UART连接蓝牙等模块
- 可检测输入（电池）电压
- 4G模块底板功能
  - 提供UART接口
  - USB接口

## 硬件设计

- 12V电源输入（XH接口）
- 5V对外电源（辅助位置）
- 3.3V芯片电源
- STM32H750VBT6主控
  - 8路PWM输出
  - 6路PWM捕获
  - 2路UART
  - 1路ADC
- 4G底板

### 电源需求

- 光流：115mA@5V
- 4G模块：1A@12V
- LED指示灯：20mA@3.3V
