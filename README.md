# 概要

## 实现梯度

1. 口罩/人脸识别
2. 衣服颜色识别
3. 语义合成模板or语音合成SDK
4. 喊话（音频放大器）
5. 无人机搭建
6. 无人机超声波避障
7. 移动网络通信+定位
8. 定点循环飞行
9. 联调
10. 脸脸距离（VO）（排队人群）
11. 停机坪识别对准（“X”字符）
12. 自动无线充电
13. 给定广场范围判断
